## Objetivo

Analisar os dados da base de dados fictícia Northwind para identificar padrões de vendas, clientes mais relevantes, produtos com melhor desempenho e oportunidades comerciais.

## Ferramentas utilizadas

- SQL
- Google Sheets / Excel
- Power BI

## Perguntas de negócio

Vendas

- Qual o faturamento total?
- Qual o faturamento mensal?
- Qual o ticket médio?
- Quantos pedidos foram feitos?

Produtos

- Quais produtos mais vendem?
- Quais produtos geram mais receita?
- Quais categorias performam melhor e pior?

Clientes

- Quais clientes mais compram?
- Quais países/cidades geram mais vendas?

Funcionários

- Quais funcionarios vendem mais?
